# builder-prog
Will be our all in one building program

# Current Functions



### help


```lua
-- Returns all available commands
builder help
```

### floor

```lua
-- Builds a floor with the specified size.
-- Uses the Item in Slot 1 as Buildingblock!
builder floor -w <width> -l <length> -m <"left"|"right">
```
